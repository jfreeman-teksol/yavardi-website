# Yavardi Website

Technology Governance for Legal Operations

## Site Structure

Static HTML site with 5 pages:
- `index.html` - Homepage
- `services.html` - How Dominion Works
- `governance.html` - How Governance Protects Your Firm
- `records.html` - Engagement Records
- `contact.html` - Request Governance Examination

## Deployment

Site is deployed on Vercel at www.yavardi.com

### To Deploy Updates:

```bash
git add .
git commit -m "Update site"
git push origin master
```

Vercel auto-deploys on push to master branch.
